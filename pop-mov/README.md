# Analyzer (アナライザー)

Tools for analyzing popular videos and tracking trends.

## Features
- Video performance analysis
- Trend tracking
- Competitive analysis
- Report generation

## Components
- `video-analyzer/`: Analyze individual video metrics
  - `analyzer.py`: Engagement rate and performance score calculator
- `trend-tracker/`: Track trends over time
  - `research/`: Weekly trend research tools (research.py, Colab versions)
- `reports/`: Generated analysis reports
  - 11 markdown reports: video comparisons, cognitive science frameworks, version reviews (v1-v5+)

## Usage
1. Collect video data from platforms
2. Run analysis tools to extract insights
3. Generate reports for decision-making
